Deploy Link - https://logic-looper-mu.vercel.app

---
## 🚀 Project Overview
Logic Looper is a highly interactive web application that delivers daily logic-based puzzles with:
- streak tracking
- scoring system
- hint management
- daily engagement mechanics
- responsive UI
- client-first architecture

The platform focuses on **daily user engagement**, **performance tracking**, and **scalable puzzle rendering systems**.

---
## ⭐ Key Features

### ✅ Daily Puzzle System
- New puzzle generated daily
- Client-side puzzle engine
- Automatic daily reset mechanism
- Minimal server dependency

### 🔥 Streak System
- Daily play tracking
- Consecutive play streak count
- Heatmap-based activity tracking
- Persistent local storage state

### 💡 Hint System
- Limited hints per day
- Client-managed hint usage
- Hint tracking with penalties

### ⭐ Scoring Algorithm
- Performance-based scoring
- Time penalty calculation
- Hint penalty deduction
- Score persistence

### 🧩 Puzzle Engine
- Universal puzzle renderer
- Modular puzzle architecture
- Binary puzzle implementation
- Number matrix puzzle support
- Scalable puzzle type system

### 🎨 Modern UI/UX
- Responsive design
- Smooth animations and transitions
- Clean layout architecture
- Tailwind CSS styling
- Glassmorphism UI elements

### 🔐 Authentication
- Firebase Google Authentication
- Protected routes
- User session handling
